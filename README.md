# AI Fake News Detector

Machine Learning based Fake News Detection System using Python and Scikit-Learn.

## Features
- Detects Fake and Real News
- Uses TF-IDF Vectorization
- Logistic Regression Model

## Technologies
- Python
- Scikit-Learn
- Pandas

## Run

Install dependencies:

pip install -r requirements.txt

Train model:

python train.py

Run application:

python app.py
